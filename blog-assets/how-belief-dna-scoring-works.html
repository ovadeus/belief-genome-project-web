The weight (0.90, 0.50, 0.30) determines how much influence your response has on each dimension. The primary dimension is the one the statement most directly measures.

The direction (+1 or -1) flips the meaning. If a statement says "free will is an illusion" and you agree, a direction of -1 means your Free Will score goes down — because you're expressing belief in determinism, which is the opposite end of that dimension.

Step 4: The Math — How Scores Accumulate

Each dimension maintains a weighted accumulator — a running tally that combines every response that touched it. Here's what happens when you submit a response:

// Your slider position (0-100%) becomes a value from 0.0 to 1.0
slider_value = 0.75 // you "likely agree"

// Normalize to -1 to +1 range
normalized = (0.75 × 2) - 1 = 0.50

// Apply direction (this probe has dir = -1 for dimension #109)
directed = 0.50 × (-1) = -0.50

// Apply quality weight (DIRECT = 1.0) × dimension weight (primary = 0.9)
effective_weight = 0.9 × 1.0 = 0.90

// Add to accumulator
accumulator.sum += -0.50 × 0.90 = -0.45
accumulator.totalWeight += 0.90
accumulator.count += 1

Over many responses, the accumulator builds up evidence. Each new response is layered on top of the previous ones, weighted by how relevant and how diagnostic it was. This is why more responses = more accurate DNA — the signal strengthens while noise averages out.

Step 5: From Accumulator to DNA Digit

When we need your final score for a dimension, we compute a weighted average, then map it to the 0-9 scale used in the DNA string:

// After 5 responses touching dimension #109:
accumulator = { sum: -1.82, totalWeight: 3.15, count: 5 }

// Weighted average (stays in -1 to +1 range)
average = -1.82 / 3.15 = -0.578

// Map from [-1, +1] to [0, 9]
dna_digit = round((-0.578 + 1) / 2 × 9) = round(1.90) = 2

Why This Matters

Traditional personality tests ask you 50 questions and give you a fixed label. The Belief Genome is different:

1. It's incremental. Every new reflection refines your scores — your DNA evolves as you do.
2. It's multi-dimensional. One response updates 2-3 dimensions at once, building a rich picture faster.
3. It's quality-weighted. A deep philosophical statement shapes your profile more than a casual preference.
4. It uses direction. The system understands that agreeing with "free will is an illusion" means you lean away from free will.
5. It's portable. Your entire worldview compresses into 136 characters you can share, compare, and track over time.
