> - `AnalyzePage.tsx` → `/genome/analyze` — Full DNA rebuild from all responses
> - `SyncPage.tsx` → `/genome/sync` — Sync status across Chrome extension, website, desktop app + manual sync

> - `POST /api/genome/analyze` — Rebuilds all 124 dimension scores from scratch
> - `GET /api/genome/sync/status` — Response counts by source (extension/web/desktop)
> - `POST /api/genome/sync` — Merges incoming responses with deduplication
>
